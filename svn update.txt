Λοιπόν, πριν μια ώρα περίπου γύρισα απ'την συνάντηση με τον κύριο Πεταλίδη στην οποία πραγματοποιήσαμε 
κάποιες σημαντικές αλλαγές στο svn για τις οποίες οφείλω να σας ενημερώσω. Κατ'αρχήν παραθέτω μια εικόνα 
με το νέο directory outlook (τελική μορφή, και ΔΕΝ θα ξανααλλάξει): http://img502.imageshack.us/img502/9527/imgis.jpg

Η πολιτική μας από δω και στο εξής είναι:
οι developers:
-δουλεύετε όλοι στο branches/NDEV/Reception/src. Εκεί ανεβάζετε τα αρχεία σας, και μετά από κάθε iterration θα γίνεται 
									merge από το MAIN για να έχουμε κι εκεί σωστό κώδικα
-στο branches/NDEV/Reception/docs μπαίνουν τυχόν documents πχ, manuals etc.


οι testers:
-δουλεύετε όλοι στο branches/NTEST/Reception/tests. Εκεί ανεβάζετε τα test units σας. 

στο MAIN θα υπάρχει ο καλύτερος κώδικάς μας και εκεί θα μεταφέρω ΜΟΝΟ ΕΓΩ αρχεία

στο tag θα γίνονται τα releases κάθε φορά που κάνουμε νέο release (εγώ κάνω τα release).


Η διαδικασία για να βγάλουμε release είναι:
1. έχουμε κώδικα στο MAIN branch που τρέχει
2. εκτελώ τα tests και ΑΝ βγουν όλα οκ
3. κάνω release

Μετά από κάθε iterration θα κάνω merge το περιεχόμενο τoυ MAIN στο /NDEC/Reception/src για να υπάρχει κι εκεί ο σωστός κώδικας...
